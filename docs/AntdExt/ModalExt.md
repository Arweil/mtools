## Modal.useModal()
主要针对zeus主题下的confirm进行了调整，新增了如下属性
| 属性 | 说明 | 类型 | 默认值 | 版本 |
| --- | --- | --- | --- | --- |
| type | 弹窗类型，主要影响弹窗的icon | info \| success \| error \| warning \| confirm | confirm |  |
| backgroundImg | 弹窗背景 | React.ReactNode | | |
